Key Features
Apple Music Inspired UI: A beautiful, dark-themed interface built with CustomTkinter that mirrors the clean aesthetic of Apple's music ecosystem.
Smart Metadata Engine: * Auto-Save on Navigation: Changes to your tags are automatically committed when you switch between tracks. No more lost edits.
Intelligent Batching: Update Album, Artist, and Genre across multiple files simultaneously, while keeping Title and Track Number unique to each file.
High-Res Album Art Support: Easily embed JPEGs or PNGs directly into your FLAC files. Selecting an image automatically applies it to every file in your current edit list.


Technical Details
Engine: Powered by Mutagen for lossless, high-speed metadata manipulation.
Framework: CustomTkinter for a modern, hardware-accelerated UI.
Distribution: Fully compatible with PyInstaller for standalone .exe creation.
